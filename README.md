# FORTIS — Sistemas de Seguridad

Landing page de marketing para **FORTIS**, empresa integradora de sistemas de seguridad electrónica. El sitio presenta los cuatro servicios principales: CCTV, videovigilancia inteligente, control de accesos biométricos e infraestructura de redes.

## Tecnologías principales

- **TanStack Start** — framework full-stack React con enrutamiento por archivos
- **Tailwind CSS v4** — estilos utilitarios con variables CSS personalizadas
- **Vite** — bundler de desarrollo
- **Netlify** — plataforma de deploy con plugin oficial para TanStack Start

## Ejecutar en local

```bash
npm install
npm run dev
```

El servidor estará disponible en `http://localhost:3000`.

Para emular todas las funcionalidades de Netlify (funciones, edge, etc.):

```bash
netlify dev
```

Disponible en `http://localhost:8888`.
